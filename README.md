# CookCamp

レシピメディア風の Web サイトトップページを想定して制作した静的サイトです。  
グローバルナビゲーション、サイドメニュー、ランキング、記事コンテンツ、広告枠などの構成で組み立てています。

## デモ
https://iris-848.github.io/CookCamp/

## 動作環境
- Google Chrome / Microsoft Edge / Safari で動作確認済み

## 使用技術
- HTML / CSS

※ 本プロジェクトは静的サイトのため、JavaScript は使用していません。

## 特徴
- グローバルナビゲーション（TOP / お問い合わせ / レシピ集 / 規約）
- サイドメニュー
  - 食材別レシピ
  - 調理時間別
- 季節のおすすめレシピランキング（サムネイル一覧）
- 店舗・料理紹介の読み物コンテンツ（画像＋本文）
- 右カラムに広告バナー枠
